Watson APIs Unity SDK

Use this SDK to build Watson-powered applications in Unity.
Sign up for an IBM Cloud Account here: http://console.bluemix.net/registration

Open README.html in a web browser for general information for the SDK and each service (under Watson Services).

Documentation can be found here: "https://watson-developer-cloud.github.io/unity-sdk/" or by selecting API Reference the Watson menu (Watson -> API Reference) in Unity.
